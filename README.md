# Maara Bakes and Sweets Website

This project is a modern responsive bakery website designed for **Maara Bakes and Sweets**, a bakery located in Tirunelveli, Tamil Nadu.

## Features
- Responsive design
- Complete menu section
- Photo gallery
- Customer reviews
- Contact information
- Google Maps location
- Call-to-action buttons

## Technologies Used
- HTML
- CSS
- JavaScript
