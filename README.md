# Maara Bakes and Sweets Website

A modern responsive bakery website for Maara Bakes and Sweets featuring menu categories, gallery, customer reviews, and contact information built using HTML, CSS, and JavaScript.

## Features
- Responsive design
- Complete menu section
- Photo gallery
- Customer reviews
- Contact information
- Google Maps location
- Call-to-action buttons

## Technologies Used
- HTML
- CSS
- JavaScript
